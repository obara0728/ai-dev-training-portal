# 実践課題一覧

## 目的

GitHub、Codex、Claude Codeを実際に使いながら、AI開発の流れを覚える。

## 課題 1: GitHubに研修用リポジトリを作る

### 目的

開発内容を保存・共有する場所を作る。

### 使用ツール

- GitHub

### やること

- GitHubにログインする
- 新しいリポジトリを作る
- READMEを作る
- 最初の変更をCommitする

### 完了条件

- `README.md` が作成されている
- `docs/requirements/requirements.md` が作成されている
- `docs/learning-logs/2026-05-25-first-log.md` が作成されている

## 課題 2: 学習ログを残す

### 目的

作業の過程をあとから見返せるようにする。

### 使用ツール

- GitHub
- ChatGPT

### やること

- `docs/learning-logs/` に今日のログを作る
- つまずいたことを書く
- 解決したことを書く
- 次にやることを書く

### 完了条件

- 学習ログに今日の作業内容が書かれている

## 課題 3: プロンプトを記録する

### 目的

AIにどんな依頼をしたかを残し、社内ノウハウにする。

### 使用ツール

- ChatGPT
- Codex
- Claude Code

### やること

- `docs/prompts/` にプロンプト記録を作る
- 入力したプロンプトを書く
- 返ってきた結果を書く
- 良かった点と改善点を書く

### 完了条件

- `docs/prompts/prompt-template.md` が作成されている
